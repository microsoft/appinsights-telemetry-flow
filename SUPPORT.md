# Support

## How to Get Help

This project uses [GitHub Issues](https://github.com/microsoft/appinsights-telemetry-flow/issues)
to track bug reports and feature requests. Please search the existing issues
before filing new ones to avoid duplicates.

**This project is provided as-is with no official Microsoft support.**
It is not covered under any Microsoft standard support program or service.

For **Azure Monitor and Application Insights** product support (not this
diagnostic tool), use your standard support channel:

- [Azure Support Plans](https://azure.microsoft.com/support/plans/)
- [Azure Monitor Documentation](https://learn.microsoft.com/azure/azure-monitor/)
- [Application Insights Documentation](https://learn.microsoft.com/azure/azure-monitor/app/app-insights-overview)
- [Microsoft Q&A — Azure Monitor](https://learn.microsoft.com/answers/tags/azure-monitor)

## Security Issues

**Do not open a GitHub issue for security vulnerabilities.** Instead, follow
the instructions in [SECURITY.md](SECURITY.md).
